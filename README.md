# HMG SermonScribe v4

**Real-Time Sermon Transcription and Church Intelligence Platform**

Zero API cost. Offline-first. Enterprise-grade. Built by HMG Gospel under HMG Concepts.

## Quick Start

1. Deploy to GitHub Pages, Netlify, Vercel, or Cloudflare Pages (all free)
2. Open in Chrome or Edge for full speech recognition support
3. Click the microphone to begin transcribing

## Features

- Real-time speech-to-text transcription (10+ languages)
- Audio recording with drag-and-drop upload
- Live congregation captions via BroadcastChannel
- Bible verse auto-detection and autocomplete
- Sermon outlines, discussion guides, and flashcards
- 12 export formats (TXT, MD, HTML, DOC, PDF, SRT, JSON, RSS, etc.)
- Social media, blog, and newsletter generation
- Analytics dashboard with charts
- Church calendar, prayer wall, digital bulletin
- Notes and devotions management
- Dark mode, high contrast, dyslexic-friendly font
- PWA installable, works offline
- Full backup and restore
- Zero API costs
