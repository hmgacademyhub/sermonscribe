About HMG SermonScribe v4

A free, enterprise-grade sermon transcription and church intelligence platform. Built by HMG Gospel under HMG Concepts with zero API costs and full offline capability.

Mission

To empower churches worldwide with affordable, accessible, and powerful tools for sermon capture, management, and repurposing - without any subscription costs.

How It Works

  • Real-time Transcription: Uses your browser's built-in Speech Recognition API - no server needed.
  • Audio Recording: Records directly to your device using the MediaRecorder API.
  • Offline Storage: All data is saved in your browser's IndexedDB - no cloud dependency.
  • Live Captions: Broadcasts to other devices using the BroadcastChannel API.
  • Zero API Cost: Everything runs in the browser. No paid services required.
