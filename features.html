Feature Breakdown

Every capability is built for the realities of Nigerian churches: low bandwidth, low budget, high impact. No paid AI API. No recurring cost. Just real tools.

🎙️1. Real-Time Speech Transcription

Powered by the native Web Speech API. Continuous listening, interim + final results, auto-restart on error, speaker labelling, and multi-language support (English, Yoruba, Igbo, Hausa).

🎙️2. Audio Backup Recording

Parallel MediaRecorder track saves a WebM audio blob alongside the transcript. If the transcript misses a word, the audio file is your safety net.

📡3. Live Congregation Captions

Open the live.html page on a projector-connected laptop. The scribe's device broadcasts the transcript via BroadcastChannel so the congregation can read captions in real time on the main screen. A QR code is also generated for congregation members to scan and follow along on their own phones.

📦4. Offline-First PWA Architecture

Installable, cacheable, works with zero internet after first load. IndexedDB stores all sermons. Service Worker caches every asset.

📖5. Smart Bible Reference Detection + Autocomplete

Regex parser detects John 3:16, Romans 8:28, etc. A 200+ verse database shows full text. While typing, scripture references autocomplete instantly. One-click inserts the full KJV text as a blockquote.

✏️6. Rich Text Editor + Sermon Templates

Formatting toolbar, structured Expository / Topical / Testimony templates, caret-aware insertion, and auto-save every 10 seconds.

🔖7. Timestamped Moment Bookmarks

Tap the 🔖 button during recording to mark important moments. Each bookmark is timestamped and inserted into the transcript. Perfect for "quote this later" or "important prophecy."

✂️8. Filler Word Removal

One-click regex stripper removes "um", "uh", "like", "you know", "basically", and other speech fillers from the transcript.

🔍9. Find & Replace / Bulk Edit

Global search and replace across the entire transcript, with optional regular expression support. Fix repeated transcription errors instantly.

🎙️10. Post-Recording Audio Upload Transcription

Drag and drop an MP3, WAV, or M4A sermon recording. The browser plays it back and transcribes it using the same speech engine — no cloud upload required.

📋11. Auto-Generated Sermon Outline

Rule-based algorithm extracts key sentences, paragraph summaries, and detected scriptures to produce a structured sermon outline in Markdown format.

❓12. Discussion Questions Generator

Template-based question generator uses detected themes and scriptures to create discussion questions for small groups and Bible study.

🙏13. Prayer Request Extractor

Pattern-matching engine detects phrases like "pray for", "let us pray", and "prayer request" and auto-extracts a prayer list from the sermon.

💬14. Shareable Quotes & Social Media Clips

Algorithm extracts the most impactful sentences (short, scripture-adjacent, theme-rich) and formats them for one-click WhatsApp sharing as sermon quotes.

🃏15. Scripture Flash Cards

Turn detected Bible verses into interactive memory cards. Tap to reveal. Save to a persistent flashcard library. Perfect for personal memorisation and church discipleship.

📖16. 5-Day Devotional Generator

Rule-based devotional generator creates a structured 5-day reading plan from any sermon, with daily scripture focus, guiding questions, action steps, and closing prayers.

📊17. Sermon Analytics Dashboard

Analytics.html shows: total sermons, total words, average length, unique themes, scripture coverage heatmap, most-quoted verses, and series breakdown. All computed locally from your IndexedDB library.

📤18. Enterprise Export Suite (8 Formats)

📝 Plain Text (.txt)

Clean text with metadata header.

📝 Markdown (.md)

For blogs, GitHub, Obsidian.

🌐 Web Page (.html)

Self-contained branded page.

📄 Microsoft Word (.doc)

Opens directly in Word/WPS.

🖨️ Print / Save as PDF

Print-optimised layout.

🎬 Subtitles (.srt)

From timestamped bookmarks.

📖 5-Day Devotional (.md)

Structured follow-up content.

💬 WhatsApp Share

One-tap to congregation.

🔗19. Shareable URL + QR Code

Entire sermon encoded into a URL hash. Recipients open it instantly. QR code generated for phone scanning.

📽️20. Presenter & Projector Mode

Clean full-screen reading view. Projector mode shows large white-on-black text for sanctuary screens. Escape to exit.

🧩21. Accessibility & Shortcuts

Ctrl+Space (record), Ctrl+S (save), Ctrl+N (new), Ctrl+P (present), Ctrl+F (find/replace), Ctrl+B (bookmark). ARIA live regions, reduced-motion, keyboard focus.

💾22. Backup & Restore

Download entire library as JSON. Restore on any device. Supports series and flashcard data.

🇳🇬23. Nigerian Context Optimisation

Under 300KB core load. WhatsApp-first. Offline-first. No app store. Multi-language speech. Zero recurring cost.

🏛️24. Custom Church Branding

Upload your church name, accent colour, logo URL, and contact. The app rethemes locally for your congregation.
