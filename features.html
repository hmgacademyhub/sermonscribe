📖 Feature Documentation

HMG SermonScribe v3 includes 50+ enterprise features. All free, offline-first, and built with zero-cost APIs.

🎙️ Core Transcription
  • Real-time speech-to-text via Web Speech API in Chrome, Edge, Safari, Samsung Internet.
  • Multi-language support (English NG/US/UK, Yoruba, Igbo, Hausa).
  • Audio backup via MediaRecorder API (saves local WebM attachment).
  • Continuous listening with auto-restart on timeout.
  • Speaker labeling (Pastor, Guest, Bible Reader, Choir).
  • Vibration feedback on start, pause, stop, and bookmarks.
  • Wake Lock API keeps screen on during recording.
  • Sermon timer with countdown alerts for pastors.
  • Screen orientation lock for recording and presentation modes.
✝️ Bible Intelligence
  • Smart reference detection auto-identifies "John 3:16" style references.
  • 500+ curated verse database in KJV, NIV, ESV, NLT, AMP, MSG.
  • Free Bible API fallback with IndexedDB caching for offline use.
  • Verse autocomplete while typing references in the editor.
  • One-click insertion of detected verses into sermon notes.
  • Cross-reference linking between scripture and sermon content.
📝 Content Generation
  • Auto sermon outline generated from headings and paragraphs.
  • Discussion questions based on themes and scriptures.
  • 5-day devotional guide exported as Markdown.
  • Prayer request extractor finds prayer patterns in text.
  • Social media quote clips with shareable WhatsApp links.
  • Scripture flashcards with reveal-tap memory feature.
  • Filler word removal (uh, um, like, you know) for clean transcripts.
  • Reading level analysis (Flesch-Kincaid grade and ease).
  • Word cloud visualization of sermon themes.
  • Text-to-speech reads sermons aloud for accessibility.
📡 Live Congregation
  • BroadcastChannel + localStorage fallback for cross-tab live captions.
  • QR code generation for congregation to open captions instantly.
  • Shareable URL to open live.html on projectors or phones.
  • Screen orientation lock for landscape projector mode.
📝 Congregation Tools
  • Fill-in-the-blank notes for interactive sermon engagement.
  • Digital bulletin builder with print-optimized layout.
  • SOAP and BREAD devotional methods with structured journaling.
  • Prayer wall for submitting and agreeing with prayer requests.
  • Sermon evaluation rubric for pastoral feedback.
📤 Export Suite
  • TXT, Markdown, HTML, Word (.doc), Print/PDF, SRT subtitles, JSON.
  • 5-Day Devotional Markdown, Podcast RSS feed.
  • WhatsApp share, Native Web Share, QR code shareable URLs.
  • Import from .txt and .md files.
  • Backup & restore as encrypted JSON archives.
📊 Analytics & SEO
  • Client-side analytics dashboard with theme word clouds, preacher distribution, scripture rankings, sermon timeline.
  • SEO meta tags, Open Graph, Twitter Cards, JSON-LD structured data.
  • sitemap.xml and robots.txt for search engine discoverability.
  • Offline HTML fallback page for service worker cache.
♿ Accessibility & Enterprise
  • High contrast mode, adjustable font size (12px–32px), keyboard-only navigation.
  • Skip links, ARIA labels, focus traps, screen-reader announcements.
  • Dark/light mode with FOUC prevention.
  • PWA installable on Android, iOS, Windows, macOS, Linux.
  • Service Worker with cache-then-network strategy and offline fallback.
  • IndexedDB storage for sermons, devotions, prayers, bulletins, notes, flashcards.
  • Custom church branding (name, color, logo, phone).
  • Keyboard shortcuts (Ctrl+S, Space, N, P, F, B, K).
