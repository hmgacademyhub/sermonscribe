Complete Feature List

40+ enterprise features. Zero API cost. 100% free tools.

Core Transcription

Real-time Speech-to-Text Core

Uses the browser's built-in Web Speech API to transcribe sermons as you speak. Supports 10+ languages.

Audio Recording Core

Records high-quality audio alongside transcription. Audio files stored locally and downloadable.

Pause and Resume Core

Pause transcription during interruptions and resume seamlessly. Timer continues tracking.

Speaker Labels Core

Automatically prefix transcribed text with speaker labels (Pastor, Minister, Evangelist, etc.).

Timestamped Bookmarks Core

Add bookmarks during recording with custom notes. Exportable as SRT subtitle files.

Bible Intelligence

Auto Scripture Detection Core

Automatically detects Bible verse references in sermon text and displays them in the sidebar.

Verse Autocomplete Core

As you type a scripture reference, a dropdown appears with matching verses from the built-in Bible database.

Online Verse Fetching Free

Fetches full verse text from the free bible-api.com service. Results cached locally for offline use.

Flashcards for Memorization New

Generates flashcards from detected scripture references. Click to reveal answers.

Live Congregation Features

Live Caption Broadcasting Core

Broadcast real-time sermon captions to other devices using the BroadcastChannel API.

QR Code Generation Core

Generate QR codes for congregation members to scan and access live captions.

Export and Share

12 Export Formats Core

Shareable URLs Core

Generate unique URLs for each sermon that can be shared via link or QR code.

Full Backup and Restore Core

Export all data as a single JSON file. Restore from backup at any time.
